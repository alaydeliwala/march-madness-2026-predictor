# March Madness 2026 Bracket Predictor

## Project Structure
```
march-madness-data/
  bracket_predictor.py          # Main script (1,341 lines)
  bracket_output_2026.txt       # Generated bracket output
  parlays_2026.txt              # Parlay recommendations
  bracket_predictor_architecture.excalidraw  # System diagram
  gen_excalidraw.py             # Diagram generator
  README.md                     # Project documentation
  data/                         # 38 CSV files (2008-2026)
```

## Key Technical Details

- **Data join key**: `TEAM NO + YEAR` across all CSV files
- **Composite rating**: 2-tier z-score weighted average (85% core efficiency + 15% adjustments)
- **Win probability**: Logistic model with sigma=3.40 (calibrated via backtesting)
- **Bayesian blend**: 85% model + 15% historical seed base rates
- **Backtest accuracy**: 73.7% R64 average across 2012-2025 (target was 72-76%)
- **Upset budget system**: R64:4, R32:2, S16:1, E8:1 max contrarian picks per round

## 2026 Bracket Results
- **Champion**: Michigan (1-seed, composite #1 at 1.543)
- **Final Four**: Duke, Houston, Arizona, Michigan
- **Key contrarian picks**: VCU over UNC, Texas A&M over St. Mary's, NC State over BYU, Santa Clara over Kentucky, Tennessee over Virginia

## Dependencies
- Python 3.10+, pandas, numpy (no ML libraries)

## Running
```bash
python3 bracket_predictor.py              # Full output to stdout
python3 bracket_predictor.py > output.txt # Save to file
```

## Important Notes
- CSVs live in `data/` subfolder -- the script resolves this via `DATA_DIR`
- The Excalidraw diagram is generated by `gen_excalidraw.py` (run it to regenerate)
- Tournament Matchups CSV encodes bracket tree: consecutive pairs in CURRENT ROUND=64 are R64 games; First Four games appear as duplicate higher-seed entries
- Historical results: ROUND column in matchups = how far team actually went (1=champion, 64=lost R64)
- 2020 is excluded from backtesting (COVID cancellation)
